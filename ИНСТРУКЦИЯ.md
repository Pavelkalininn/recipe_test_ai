# Чат - Инструкция по установке

Полностью рабочее чат-приложение с авторизацией и возможностью установки на телефон как приложение (PWA).

## Что уже готово

- Авторизация по email и паролю
- Real-time чат (сообщения появляются мгновенно у всех)
- База данных PostgreSQL (Supabase)
- PWA - можно установить на телефон
- Адаптивный дизайн для всех экранов

### Альтернатива: Netlify

1. Иди на https://netlify.com
2. Войди через GitHub
3. Перетащи папку `dist` (после `npm run build`)
4. В настройках добавь те же переменные окружения
5. Готово!

## Как установить на телефон

### Android (Chrome/Edge)
1. Открой сайт в Chrome
2. Нажми три точки → "Установить приложение"
3. Готово! Иконка появится на главном экране

### iPhone (Safari)
1. Открой сайт в Safari
2. Нажми кнопку "Поделиться" (квадратик со стрелкой)
3. Выбери "На экран Домой"
4. Готово!

## Как пользоваться

1. Открой приложение
2. Создай аккаунт (email + пароль)
3. Войди
4. Пиши сообщения в общий чат
5. Все пользователи видят сообщения в реальном времени

## Технические детали

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Real-time + Auth)
- **Хостинг**: Vercel/Netlify (бесплатно навсегда)
- **PWA**: Service Worker + Web Manifest

## База данных

База уже настроена и готова к работе:
- Таблица `messages` для сообщений
- Настроены правила безопасности (RLS)
- Real-time подписки для мгновенных обновлений

## Если что-то не работает

1. Проверь, что переменные окружения добавлены правильно
2. Убедись, что сайт открыт через HTTPS (для PWA)
3. Очисти кэш браузера
4. Попробуй в режиме инкогнито

## Команды для локального запуска

```bash
npm install          # Установить зависимости
npm run dev          # Запустить локально (http://localhost:5173)
npm run build        # Собрать для продакшена
npm run preview      # Предпросмотр продакшен-версии
```
